# pyocnndata
Data for publication and open access

The repository stores data of the publication 
"Digital-analog hybrid matrix multiplication processor for optical neural networks."

# Prequisitions
## Python package dependencies
* numpy
* pandas
* scipy
* matplotlib
* scikit-learn
* scikit-image
* openpyxl
* jupyterlab
* ipywidgets
